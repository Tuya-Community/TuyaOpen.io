关于 TuyaOpen

一套开源 SDK，覆盖全芯片

TuyaOpen 提供跨芯片平台的统一应用框架——一次编写，部署到任意支持的硬件。

  • 统一 API 层屏蔽芯片差异，应用代码跨平台复用
  • 多平台支持：T2、T3、T5、ESP32、Raspberry Pi 4/5 等
  • 模块化组件设计：联网、安全、OTA、外设即插即用
  • 云端无关连接层，深度集成涂鸦 IoT Core
  • AI Agent 能力与 APP 能力快速集成，开箱即用
  • 开源 Apache 2.0 协议，代码完全透明可审计

立即开始构建
