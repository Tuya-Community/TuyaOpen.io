# 维护与发布

本页面提供关于TuyaOpen维护计划、发布周期、平台支持和开发路线图的综合信息。

## 维护计划

### TuyaOpen核心平台

TuyaOpen遵循结构化的维护计划，确保可靠性和持续改进：

- **活跃开发**：持续开发，每日提交代码
- **错误修复**：关键错误在24-48小时内解决
- **安全更新**：安全补丁在发现后72小时内发布
- **功能更新**：新功能每两周发布一次
- **主要发布**：每季度发布包含重要新功能的主要版本

### 多平台支持

我们的多平台方法确保在不同操作系统上的一致体验：

| 平台 | 维护频率 | 支持级别 |
|------|----------|----------|
| **Linux** | 每日更新 | 完全支持 |
| **Windows** | 每周更新 | 完全支持 |
| **macOS** | 双周更新 | 测试版支持 |

## SDK发布

### Arduino SDK

TuyaOpen Arduino SDK为基于Arduino的项目提供无缝集成：

- **发布频率**：每月稳定版本发布
- **测试版发布**：每周测试版发布用于测试
- **LTS版本**：每6个月发布长期支持版本
- **兼容性**：支持Arduino IDE 1.8.x和2.x

**最近发布：**
- v2.1.3（最新稳定版）- 2024年12月
- v2.2.0-beta.1（测试版）- 2025年1月
- v2.0.5（LTS）- 2024年6月
