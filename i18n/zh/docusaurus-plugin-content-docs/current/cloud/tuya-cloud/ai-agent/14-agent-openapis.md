智能体开放接口让你以开放的方式将涂鸦智能体集成到自有平台或软件产品。当你在涂鸦开发者平台开发了智能体，并希望从自己的产品中使用它时，可通过平台提供的 OpenAPI 接口访问智能体的数据。

平台智能体开放能力正在持续完善中，你可关注平台开放文档获取最新资料。

## 使用方式

智能体开放接口的访问管理基于涂鸦云开发平台。你需要先在 [云开发平台](https://platform.tuya.com/cloud) 创建一个云项目并订阅相关服务，才能正常调用。智能体接口通过云项目颁发的访问密钥进行鉴权。

## 费用说明

智能体接口暂不收取功能费用。

接口调用产生的智能体 Token 消耗将正常计量计费。计费规则详见 [智能体计量计费](https://developer.tuya.com/cn/docs/iot/ai-agent-price?id=Kegb2s2shaj4d#title-3-%E6%A8%A1%E5%9E%8B%E8%B4%B9%E7%94%A8)。
