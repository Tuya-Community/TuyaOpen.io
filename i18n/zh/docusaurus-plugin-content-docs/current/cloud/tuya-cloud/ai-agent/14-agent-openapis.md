# **智能体开放接口**
# **概述**

智能体开放接口为您提供通过开放的方式对智能体进行集成的能力。当您在涂鸦开发者平台开发了智能体，同时拥有自己的平台或者软件产品，并想要将二者进行扩展集成时，您可以通过平台提供的 OpenAPI 接口对智能体的数据进行访问。

平台智能体开放能力正在持续完善中，您可关注平台的开放文档获取最新的资料。

# **使用方式**

智能体开放接口的访问管理基于涂鸦云开发平台。您需要在 [云开发平台](https://platform.tuya.com/cloud) 创建一个云项目并订阅相关的服务，才能进行正常的调用。智能体接口通过云项目颁发的访问秘钥进行访问鉴权。

# **费用说明**

智能体接口暂时不收取功能费用。

由于接口调用产生的智能体 Token 消耗，正常计量计费。计费规则详见 [智能体计量计费](https://developer.tuya.com/cn/docs/iot/ai-agent-price?id=Kegb2s2shaj4d#title-3-%E6%A8%A1%E5%9E%8B%E8%B4%B9%E7%94%A8)。
