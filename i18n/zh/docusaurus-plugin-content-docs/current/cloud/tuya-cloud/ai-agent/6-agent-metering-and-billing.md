智能体的费用来自两部分：所运行的语言模型，以及所使用的智能语音。本文介绍各计费项、计费方式及当前单价。

## 计费项

智能体的计费项包括 **模型费用** 和 **智能语音费用**。

## 计费周期

各计费项均按天计量，按实际使用量结算。账单通常在当前计费周期结束后一天生成，具体出账时间以系统为准。

## 模型费用

### 模型类型

如何区分模型类型，请参考 [可用模型](https://platform.tuya.com/exp/model)。

### 计费公式

模型服务按 Token 使用量计费：

**模型费用 = Token 使用量 × 模型 Token 单价**

Token 是语言模型处理文本的基本单位。模型将输入文本分解成一系列 Token，再进行处理和分析。Token 可以是单词、字符、子词片段或其他文本片段，具体划分方式取决于模型使用的分词算法，因此 Token 的计算和处理方式会随模型架构和设计而不同。

## 智能语音费用

### 计费公式

智能语音分为语音输入（ASR）和语音输出（TTS）两部分：

**智能语音费用 = ASR 单价 × 输入音频时长 + TTS 单价 × 输出音频字符数**

- **自动语音识别（ASR）**：识别并理解人类自然语音输入的技术，通过分析处理语音信号，将音频转化为文本。
- **文本转语音（TTS）**：将文本转换成语音输出的技术，模拟人类朗读，将文字信息转变为音频。
