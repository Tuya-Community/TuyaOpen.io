涂鸦 [智能体开发平台](https://platform.tuya.com/exp/ai) 让您基于大模型搭建 AI 智能体，并借助智能体管理智能设备。您在平台上配置智能体，然后将其发布到各应用端和设备，或通过 API 和 SDK 集成到自己的业务系统。

## 为什么选择涂鸦智能体开发平台

- **一站式开发平台**：所需的功能和配套能力均已内置，无需额外开发即可直接使用。
- **AI 幻觉控制**：在物联网领域，AI 幻觉可能导致严重后果。平台通过批量测试、工作流和检索增强生成（RAG）来加以解决。
- **涂鸦物模型智能体**：智能体直接与设备交互，省去将大模型接入设备控制的复杂开发。
- **多模型协作**：平台将大模型与涂鸦积累的小模型相结合，为设备控制带来更快的响应、更高效的交互和更低的算力成本。
- **开放可扩展**：支持二次开发，您可以随产品需求扩展智能体的能力。
