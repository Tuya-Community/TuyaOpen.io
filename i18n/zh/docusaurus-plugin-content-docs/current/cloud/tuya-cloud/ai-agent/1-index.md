# **智能体开发**

涂鸦 [智能体开发平台](https://platform.tuya.com/exp/ai) 是一款提供 AI 解决方案的平台系统。您可以在平台上快速搭建基于大模型的 AI Agent，进而实现智能设备的高效管理。平台支持发布 AI 应用至各应用端，或通过 API 和 SDK 集成至业务系统。

# **为什么选择涂鸦智能体开发平台？**

- **一站式开发平台**：涂鸦提供全面的功能和配套能力，无需额外开发，直接在平台使用。
- **AI 幻觉解决方案**：在物联网领域，AI 幻觉问题可能会导致严重后果。为了有效解决这一问题，涂鸦提供批量测试、工作流、RAG 等多种方案。
- **涂鸦物模型智能体**：智能体具备与设备直接交互的能力，能够简化大模型控制设备的复杂开发过程。
- **多模型协作**：平台与涂鸦多年来积累的小模型相结合，能够为设备控制提供更快捷的体验、更高效的用户交互、以及更低的算力成本。
- **强大的开放性**：支持二次开发，您可根据需求灵活扩展智能体功能。
