# **添加自定义音色**
# **概述**

本文为您介绍如何为智能体添加自定义音色，请完成以下步骤：

- 第一步：[声音来源的版权或原创性证明](https://developer.tuya.com/cn/docs/iot/ai-agent-customsounds?id=Kf6gftgfduuqt#claim)。
- 第二步：[签署不侵权承诺函](https://developer.tuya.com/cn/docs/iot/ai-agent-customsounds?id=Kf6gftgfduuqt#commit)。
- 第三步：[提交上述材料及用于复刻的声音文件](https://developer.tuya.com/cn/docs/iot/ai-agent-customsounds?id=Kf6gftgfduuqt#submit)。

# **声音来源的版权或原创性证明**

请选择以下任一方式进行版权或原创性自证：

### 方式一：声音 IP 的当地版权登记

对于自身持有版权的开发者，需要在该声音作品的创作地区或售卖地区做版权登记。因版权登记时间较长，可以给涂鸦提交登记的申请证明材料（比如提交申请的官网截图或受理通知书等），证书可后补。

### 方式二：完整声音 IP 授权链路

非自身持有版权的开发者，需提供下列文件：

- 提供原声版权所有人给开发者的授权合同，[详见参考附件](https://drive.weixin.qq.com/s?k=AGQAugfWAAkXX4rKLm)；若中间还有其他转授权情况，需要提供完整的各段授权证明。
- 注册并登录 [知识产权保护平台](https://ipr.tsa.cn/time/ApplyTime)，如图所示，上传声音文件及声明录制证明材料，获得原创时间戳证明证书。
    
    ![添加自定义音色](https://images.tuyacn.com/content-platform/hestia/1765270378769a8e6e8fe.png)
    

# **签署不侵权承诺函**

签署并回传 [承诺函文件](https://drive.weixin.qq.com/s?k=AGQAugfWAAkZhA0ME3)。

# **提交上述材料及用于复刻的声音文件**

暂不支持自助上传，请联系与您对接的涂鸦工作人员协助处理。为保证审核效率，请开发者在提交材料和文件前自查下列文件，并确保其准确性、一致性：

- 版权或原创证明文件内容正确。
- 免责协议内容正确填写，客户签章正确。
- 证明文件中提供的音频试听与提交的音频文件内容一致。
- 提供对应的音频信息用户复刻音色。
- 中国区：WAV 格式，音频文件，时长 20 秒左右，24 K 或 48 K 采样率，单声道。
- 海外区：WAV 格式，音频文件，时长小于 10 秒，24 K 或 48 K 采样率，单声道。海外区需注意录音的内容要求：根据对应语种说出 [此文档](https://cloud.google.com/text-to-speech/docs/chirp3-instant-custom-voice?hl=zh-cn) 中提供的内容。

确认以上信息后，使用 [此增值服务](https://www.tuya.com/vas/commodity/TIMBRE_CLONE) 下单。
