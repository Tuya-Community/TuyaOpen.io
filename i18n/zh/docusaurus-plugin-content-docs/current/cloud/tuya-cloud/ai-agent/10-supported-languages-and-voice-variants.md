# **智能体支持语种及音色**
# **ASR、TTS 说明**

AI 对话功能的实现依赖两项核心技术：自动语音识别（Automatic Speech Recognition，ASR）和 语音合成（Text-to-Speech，TTS）。

简单来说，ASR 负责把用户输出的语音转化成文字，而 TTS 则为把智能体的回复转化成语音并读出来。只有当这两项技术都支持某种语言时，智能体才能实现真正 “能听会说” 的语音交互。

目前，智能体开发平台已支持多个国家和地区的语种 ASR 和 TTS，使得用户能够用本地语言与智能体进行自然对话。例如，在中国，已支持普通话、英文、粤语以及部分地方方言（如四川话、河南话等）；在海外，已支持英语、西班牙语、日语、法语等主流语言。

建议优先选择下表中的 **已支持语种**，以确保用户能顺畅进行语音输入和语音输出。对于暂未支持的语种，请等待系统后续支持。
