智能体开发平台支持一组固定的语种与音色用于语音输入和输出。本文列出各地区支持的语种及音色数量，帮助您选择端到端可用的组合。

## ASR 与 TTS 说明

AI 对话功能依赖两项核心技术：自动语音识别（Automatic Speech Recognition，ASR）和语音合成（Text-to-Speech，TTS）。ASR 负责把用户的语音转换成文字，TTS 则把智能体的回复转换成语音并读出。只有当这两项技术都支持某种语言时，智能体才能实现真正 “能听会说” 的语音交互。

目前智能体开发平台已支持多个国家和地区的语种 ASR 和 TTS，使用户能够用本地语言与智能体自然对话。例如，在中国已支持普通话、英文、粤语以及部分地方方言（如四川话、河南话等）；在海外已支持英语、西班牙语、日语、法语等主流语言。

:::tip
建议优先选择下表中的已支持语种，以确保语音输入与输出顺畅。对于暂未支持的语种，请等待系统后续支持。
:::
