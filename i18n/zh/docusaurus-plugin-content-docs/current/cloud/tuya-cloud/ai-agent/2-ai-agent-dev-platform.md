# **智能体开发平台**
# **概述**

本智能体开发平台集成了多种语言模型，旨在为用户提供高效而灵活的智能体管理功能。用户可以通过配置和调试，轻松部署和运行智能体相关应用。

# **创建 Agent**

1. 首先，使用您的账号登录 [涂鸦开发者平台](https://platform.tuya.com/)。
2. 在平台首页的 **开发者工作台** > **AI 智能体** 下单击 **进入 AI 智能体开发**，或单击左侧菜单栏上的 **智能体** > **智能体开发** > **我的智能体**，然后进入 [智能体开发平台](https://platform.tuya.com/exp/ai)。
    
    ![智能体开发平台](https://images.tuyacn.com/content-platform/hestia/1748341742cfe079c84cc.png)
    
    ![智能体开发平台](https://images.tuyacn.com/content-platform/hestia/1748343220b3b1fdde27c.png)
    
3. 单击页面右上角的 **创建 Agent**，然后在 **新增 AI 项目** 窗口完成如下图所示的配置，配置完成后单击 **确定**。
    
    ![智能体开发平台](https://images.tuyacn.com/content-platform/hestia/17442551519e907e6e9ce.png)
    
4. 然后，进入智能体的开发页面，配置模型能力、提示词等，具体说明请参考后文。
    
    ![智能体开发平台](https://images.tuyacn.com/content-platform/hestia/1744263692ffd60061abb.png)
    

# **模型能力配置**

### 基础配置

![智能体开发平台](https://images.tuyacn.com/content-platform/hestia/1733383040da84e78c213.png)

**使用的 LLM 模型**

选择智能体所用的大语言模型。由于合规原因，不同的数据中心提供的语言模型各不相同，且不同语言模型的 Token 消耗费用也存在差异。

**记忆消息数**

配置智能体能够记住和处理的先前对话或交互的数量。这些历史消息帮助智能体保持上下文连续性，从而提供更相关和准确的响应。当然历史消息数量越多，每次对话所消耗的算力也会越大，可以根据需求场景配置历史消息数。

### 技能配置

![智能体开发平台](https://images.tuyacn.com/content-platform/hestia/173338306831b9e30ac1e.png)

**插件**

插件是指可以添加到智能体中的独立功能。插件功能允许智能体调用各种工具或 API，例如设备查询、设备控制和场景控制等。插件扩展了智能体的能力，使其能够执行更多样化和复杂的任务。

**知识库**

知识库是指用于存储和管理信息、数据和知识的系统，智能体可以从中获取答案或指引，以便更好地回答用户的问题或执行任务。可以在知识库中存储包含各种形式的信息，例如文本、文档、图像和视频等。

知识库采用了检索增强生成（Retrieval-Augmented Generation，RAG）技术，用于增强智能体的回答质量。RAG 的工作原理为：

1. 信息检索。首先，智能体通过检索模型或算法，会从知识库中检索与用户查询相关的文档或信息片段。
2. 答案生成。在获取相关文档后，智能体会根据上下文和用户查询，生成自然和连贯的回答。

**工作流**

如果遇到特定场景，要让智能体根据指定的逻辑去实现相关任务或回复用户，就需要让智能体基于设置的工作流来执行。

# **提示词开发**

![智能体开发平台](https://images.tuyacn.com/content-platform/hestia/1733383113b379c1e8eac.png)

在 **Prompt** 框中，输入用于引导语言模型生成对话或执行任务的提示词。在输入提示词前，建议先查看 [Prompt 文档](https://www.tuyaos.com/viewtopic.php?t=3724)，了解 Prompt 原则。在输入提示词后，还能够优化提示词，以帮助智能体更有效地落地业务。

# **调试与发布**

### 调试

![智能体开发平台](https://images.tuyacn.com/content-platform/hestia/1733383283ed7ff50a368.png)

1. 配置保存后，单击 **获取调试二维码**。
2. 可以用 **智能生活** App 扫描二维码进行测试，确保配置的正确性和功能的正常运行。

在发布到线上前，建议进行充分测试，确保系统功能正常运行。

### 发布

调试完成后，单击页面顶部的 **发布**，将配置发布到线上环境，使其生效。

发布后，发布记录将显示之前发布的所有版本信息，便于用户跟踪和管理配置历史。

# **应用配置**

1. 单击页面右上角 **应用管理**。
    
    ![智能体开发平台](https://images.tuyacn.com/content-platform/hestia/1733388500a210c106ef7.png)
    
2. 在出现的页面里，配置智能体在应用端的交互形态。
    
    ![智能体开发平台](https://images.tuyacn.com/content-platform/hestia/1726284048fbee95842bd.png)
    

### 选择应用端载体

用户可选择应用端载体类型，包括 App、云云对接、音响、SaaS 和中控屏。

### 智能体展示名称

用户可以设置智能体展示名称，以便在应用中显示。

### 欢迎语

用户可以输入自定义的欢迎语，在用户打开应用时显示。

### 浅色模式

- 主题色：用户可以选择浅色模式下的主题颜色。
- 背景色：用户可以选择浅色模式下的背景颜色。
- 会话框背景色：用户可以选择浅色模式下会话框的背景颜色。

### 深色模式

- 深色主题色：用户可以选择深色模式下的主题颜色。
- 深色背景色：用户可以选择深色模式下的背景颜色。
- 深色会话框背景色：用户可以选择深色模式下会话框的背景颜色。

### 背景图片

上传背景图片：用户可以上传自定义背景图片，支持格式为 JPG、JPEG 和 PNG，大小不超过 3 MB。

### 保存

当应用端载体为 App 时，保存后，会在对应 App 中根据上述配置自动生成智能体对话小程序，且扫码调试功能也将根据配置信息展示交互。

# **关联产品与智能体**

为进一步开发产品的 AI 功能，可以将产品与智能体进行关联。更多详情，请参考 [产品 AI 功能开发](https://developer.tuya.com/cn/docs/iot/AI-feature?id=Keapy1et1fc63)。
