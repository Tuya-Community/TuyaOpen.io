title: Chat with Agent
---

The Chat with Agent API retrieves the chat history of a specified agent.

## API list

| API | Description |
| --- | --- |
| [Query Chat History](https://developer.tuya.com/en/docs/iot/ai-agent-api-Query-chat-history?id=Ketypb0skxybt) | Query an AI agent's chat history by device ID, bound role type, and role ID. |
