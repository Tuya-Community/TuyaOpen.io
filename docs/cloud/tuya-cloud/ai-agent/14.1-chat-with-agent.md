# **Chat with Agent**

This API enables you to get the chat history with the specified agent.

# **API list**

| API | Description |
| --- | --- |
| [Query Chat History](https://developer.tuya.com/en/docs/iot/ai-agent-api-Query-chat-history?id=Ketypb0skxybt) | Query the chat history of an AI agent based on the device ID, bound role type, and role ID. |
